# 🌱 PlantParadise - Professional Plant E-commerce Store

A modern, fully-featured e-commerce web application for plant enthusiasts. Built with vanilla JavaScript, featuring real plant images, complete shopping cart functionality, and professional design.

## 🚀 Getting Started

### Prerequisites
- Modern web browser
- Text editor (VS Code recommended)
- Local server for development (Live Server extension)

### Installation
```bash
# Clone the repository
git clone https://github.com/archashqer/houseplant-store.git
cd houseplant-store

# Open with Live Server or serve locally
# No build process required - pure HTML/CSS/JS
```

