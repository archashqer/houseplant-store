# 🌱 PlantParadise - Premium Houseplant Store

A modern, responsive e-commerce web application for houseplant enthusiasts. Built with React and Redux Toolkit, featuring a complete shopping cart experience and beautiful plant catalog.

## ✨ Live Demo

🌐 **Website:** [View Live Store](https://archashqer.github.io/houseplant-store)
📦 **Source Code:** [GitHub Repository](https://github.com/archashqer/houseplant-store)

## 🚀 Getting Started

### Prerequisites
- Node.js (version 14 or higher)
- npm or yarn package manager
- Modern web browser

### Installation & Setup
```bash
# Clone the repository
git clone https://github.com/archashqer/houseplant-store.git
cd houseplant-store

# Install dependencies
npm install

# Start development server
npm start

# Build for production
npm run build

# Deploy to GitHub Pages
npm run deploy
```

## 🌍 About PlantParadise

We believe in bringing the beauty of nature indoors while supporting sustainable living practices. Each plant in our carefully curated catalog is selected for its air-purifying qualities, ease of care, and ability to transform living spaces.

Our mission is to make indoor gardening accessible to everyone, whether you're a complete beginner or an experienced plant parent. We provide not just beautiful plants, but also the knowledge and support needed to help them thrive in your home.
