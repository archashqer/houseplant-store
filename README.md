## ğŸš€ Getting Started

### Prerequisites
- Node.js (version 14 or higher)
- npm or yarn package manager
- Modern web browser

### Installation & Setup
```bash
# Clone the repository
git clone https://github.com/archashqer/houseplant-store.git
cd houseplant-store

# Install dependencies
npm install

# Start development server
npm start

# Build for production
npm run build

# Deploy to GitHub Pages
npm run deploy
```
